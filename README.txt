HƯỚNG DẪN

1. Cài đặt python 3.5.3

2. Cài đặt các package cần thiết:
>> python3 -m pip install -r requirements.txt

3. Chạy thuật toán
3.1.a Xác định sự tồn tại bao phủ rào chắn
>> python3 main.py --mode=0

3.1.b Tối thiểu hóa chi phí xây dựng một rào chắn
>> python3 main.py --mode=1

3.2.  Tối đa số rào chắn đạt được
>> python3 main.py --mode=2
